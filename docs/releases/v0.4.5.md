# Pi-Vis v0.4.5

## Highlights

- Added workspace-wide saved-session search with **Cmd/Ctrl+Shift+F**:
  - Search session names and persisted conversation content across the active workspace and its worktrees.
  - Browse snippets and highlights in a read-only preview before opening a result.
  - Opening a result uses the normal session lifecycle and preserves saved branches and session context.
- Expanded worktree workflows:
  - Create a fresh worktree and branch for a new session, or attach an existing worktree.
  - Switch established sessions between the workspace and worktrees from the session header.
  - Validate worktree paths, preserve conversation and draft state, and carry local changes into newly created worktrees.
- Added commit-range selection to the diff viewer for reviewing changes across a selected span of commits.
- Improved live-session reliability across reloads, interruptions, compaction, navigation, queued prompts, and extension panels. Interrupted transcript history is now preserved instead of disappearing.

## Fixes and polish

- Fixed worktree detection, persistence, relaunch recovery, branch switching, and nested-workspace handling.
- Fixed composer focus, admission, draft clearing, and queued-message restoration in edge cases.
- Improved authority synchronization and recovery when renderer, host, or panel state is interrupted or rebuilt.
- Improved Escape handling and panel rendering for unified TUI and extension UI.
- Fixed packaged builds so the bundled Pi runtime and session-search worker start correctly.

## Runtime and testing

- Pi-Vis now ships and uses the pinned **Pi 0.80.10** runtime instead of detecting a separate Pi installation for production sessions.
- CI and the release suite now build the packaged app and run render and Electron end-to-end coverage, including real SDK-host compatibility, session search, worktree switching, transcript recovery, and authority-fault journeys.

## Upgrade notes

- No manual action required. Existing sessions and worktree associations are revalidated automatically.

## Assets

- macOS Apple Silicon ZIP for installer/auto-update
- macOS Apple Silicon DMG for manual install
