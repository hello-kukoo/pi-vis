# Pi-Vis v0.5.0

## Highlights

- Added native provider sign-in flows, including `/login` support and a provider-login dialog for authenticating from Pi-Vis.
- Added extension update management with background update checks, visible installed versions, and update status in Settings.
- Improved worktree workflows, including clearer worktree controls and more reliable switching and attachment behavior.
- Added copying of selected conversation-tree entries, matching Pi's copy eligibility rules.
- Improved commit-range selection and diff editing workflows.
- Unified tool-card inspection and refined tool output presentation, activity indicators, and transcript layout.

## Fixes and polish

- Improved session, panel, and transcript synchronization during lifecycle transitions and reloads.
- Refined composer, sidebar, picker, settings, worktree, diff, and tool-card alignment and spacing.
- Improved model catalog refresh behavior and extension update handling.
- Fixed assorted tree-copy, worktree, tool-output, activity-indicator, and UI consistency issues.

## Runtime and testing

- Expanded unit, render, and E2E coverage for provider login, extension updates, worktree switching, tree copy, tool output, transcript history, and lifecycle races.

## Upgrade notes

- No manual action required. Existing sessions and saved state continue to work normally.

## Assets

- macOS Apple Silicon ZIP for installer/auto-update
- macOS Apple Silicon DMG for manual install
